## 🎮 **Unleash the Beast** 😈  
### **Quick Start**  
```bash
git clone https://github.com/BhanuGuragain0/Hacker_AI.git
cd Hacker_AI
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python launcher.py --mode full
```

### **Modes**  
- **CLI**: `python launcher.py --mode cli` ⌨️  
- **Voice**: `python launcher.py --mode voice` 🎙️  
- **AR**: `python launcher.py --mode ar` 🕶️  
- **API**: `python launcher.py --mode api` 🌐  
- **Web UI**: `cd frontend/web_ui && npm run dev` 🖼️  

### **Hack Commands**  
- **Scan**: `frontend/cli/cli_main.py scan --target 192.168.1.0/24 --ai --quantum` 🔍  
- **Exploit**: `operations/exploit_framework/exploit_generator.py --template ransomware` 💣  
- **Voice Recon**: `frontend/terminal/voice_commands/processor.py recon --target example.com` 🎤  
- **AR Visualize**: `frontend/terminal/ar_visualizer/visualizer.py visualize --topology results.json` 🕶️
