## 🛠 **How It’s Built** 🧱  
Hacker_AI’s architecture is a modular, quantum-fueled monster designed for red team dominance.

### **Architecture Diagram**  
```mermaid
graph TD
  A[User Interfaces<br>CLI, Voice, AR, Web, API] --> B[API Gateway<br>api/web_api_router.py]
  B --> C[Microservices<br>backend/services/]
  C --> D[AI Models<br>core/ai/models/]
  C --> E[Quantum Engine<br>core/ai/models/quantum/]
  C --> F[Database<br>data/db/]
  C --> G[Cache<br>backend/cache/]
  C --> H[Security Modules<br>core/security/]
```

### **Tech Stack** ⚙️  
- **AI/ML**:  
  - **Models**: Gemma 3 (`core/ai/models/pre_trained/gemma_3/`), QWQ 32B (`core/ai/models/pre_trained/qwq_32b/`), DeepSeek Coder V2 (`core/ai/models/pre_trained/deepseek_coder_v2_lite_instruct/`) 🤖  
  - **Tools**: PyTorch, Qiskit (`core/ai/models/quantum/quantum_entropy.py`), DGL, Transformers ⚡  
- **Voice & AR**:  
  - **Voice**: `speechrecognition`, `pyaudio` (`terminal/voice_commands.py`) 🎙️  
  - **AR**: Three.js (`frontend/src/lib/`), TypeScript (`frontend/src/components/visualizer.ts`) 🕶️  
- **Backend**:  
  - **Framework**: FastAPI (`api/web_api_router.py`) 🌐  
  - **Storage**: Redis (`backend/cache/redis_cache.py`), PostgreSQL (`data/db/database.py`) 💾  
- **Deployment**:  
  - **Container**: Docker (`infrastructure/deployment/docker/Dockerfile`) 🐳  
  - **Orchestration**: Kubernetes (`infrastructure/deployment/kubernetes/deployment.yaml`) ☸️

## 🔄 **How It Works** ⚔️  
Hacker_AI operates in a 7-phase workflow for maximum devastation:  

1. **🛠️ Initialization**:  
   - Loads configs (`utils/config_loader.py`) and models (`core/ai/models/`) ⚙️  
2. **🕵️‍♂️ Reconnaissance**:  
   - Gathers intel with `operations/recon/active_recon.py` 🔍  
   - Quantum ML recon via `core/ai/models/quantum/quantum_ml_recon.py` ⚛️  
3. **💥 Exploitation**:  
   - Deploys payloads via `core/operations/exploit_framework/exploit_delivery.py` 💣  
4. **🐝 Swarm Assault**:  
   - Syncs agents in `backend/agents/agent_orchestrator.py` 🤖  
5. **📡 Monitoring**:  
   - Tracks ops with `backend/core/analytics/performance_analytics.py` 📊  
   - Visualizes in AR with `terminal/ar_renderer.py` 🕶️  
6. **🛡️ Self-Healing**:  
   - Recovers via `core/self_healing/self_healer.py` ⚕️  
7. **📈 Evolution**:  
   - Learns with `core/ai/trainers/federated_trainer.py` 🧠

## 🎮 **Usage** 😈  
### **Quick Start**  
```bash
git clone https://github.com/BhanuGuragain0/Hacker_AI.git
cd Hacker_AI
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python launcher.py --mode full
```

### **Modes**  
- **CLI**: `python launcher.py --mode cli` ⌨️  
- **Voice**: `python launcher.py --mode voice` 🎙️  
- **AR**: `python launcher.py --mode ar` 🕶️  
- **API**: `python launcher.py --mode api` 🌐  
- **Web UI**: `cd frontend/web_ui && npm run dev` 🖼️  

### **Examples**  
- **Scan**: `frontend/cli/cli_main.py scan --target 192.168.1.0/24 --ai --quantum` 🔍  
- **Exploit**: `core/operations/exploit_framework/exploit_delivery.py --template ransomware` 💣  
- **Voice Recon**: `terminal/voice_commands.py recon --target example.com --quantum` 🎤  
- **AR Visualize**: `terminal/ar_renderer.py visualize --topology results.json` 🕶️
